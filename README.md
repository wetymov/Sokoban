### Sokoban Play

#### Упралевние
| Кнопка |       Действие        |
|:------:|:---------------------:|
| W      |    Движение вверх     |
| A      |    Движение влево     |
| S      |     Движение вниз     |
| D      |    Движение вправо    |
| R      | Перезагрузить уровень |
